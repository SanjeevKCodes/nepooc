# NepOOC: Nepali Political Misinformation Detection

Official code for the paper: **"NepOOC: Multimodal OOC Misinformation Detection in Low-Resource Nepali Political Media"**.

## 📁 Folders
- `src/`: Training scripts for our 5 models.
- `eval/`: Scripts to make the tables for the paper.
- `results/`: The scores (JSON files) from our tests.

## 🚀 How to use
1. Run the scripts in `src/` using Kaggle.
2. Download the `results/` folder.
3. Run `eval/nepooc_eval_master.py` to see the final tables!
